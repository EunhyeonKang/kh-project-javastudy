CSS선택자

선택자란 특정한 HTML태그를 선택할 떄 사용하는 기능, 태그를 선택해서 원하는 스타일과
기능을 정의할 수 있음

모든 선택자 (*)

HTML태그 내부의 모든 태그를 선택할 수 있음.

태그 선택자

HTML문서 내부의 같은 태그를 모두 선택할 때 사용

아이디 선택자 (#)

HTML문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택

아이디는 고유하게 지정하는 것을 원칙으로 함

  1. 아이디 선택자 테스트1
  2. 아이디 선택자 테스트2
  3. 아이디 선택자 테스트3
  4. 아이디 선택자 테스트4
  5. 아이디 선택자 테스트5

클래스 선택자 (.)

HTML 문서 내에 해당하는 클래스 속성의 값을 가진 태그만 선택

아이디와 다르게 여러개 중복해서 사용 가능

  • 클래스 선택자 테스트1
  • 클래스 선택자 테스트2
  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5
