BOM(Browser Object Model)

window객체

window.open() : 높이와 너비를 입력하여 새창을 띄어줌

setTimeout(함수,시간(ms)) : 일정시간 후 함수를 한번 실행 id값 리턴

clearTimeout(id) : settimeout()실행을 종료

setInterval(함수,시간(ms)) : 일정 시간마다 함수를 반복하여 실행 id값 리턴

clearInterval(id) : setInterval()실행을 종료

window.onloar : 윈도우 객체가 로드가 완료되면 자동으로 실행되는 함수(윈도우 객체 로드 완료 : 모든 태그가 화면에 나타나면)

onloadTest

location : 브라우저 주소표시줄과 관련된 객체

navigator : 브라우저 정보에 대한 객체
