자바스크립트 배열

배열 선언

배열 또한 하나의 자바스크립트 객체로 기본적으로 사용가능한 함수 존재

indexOf(값) : 배열에서 해당하는 값이 위치한 인덱스번호 리턴

concat(배열명) : 매개변수로 받은 배열과 결합하여 리턴

join("구분기호") : 배열을 구분기호를 이용하여 결합하여 하나의 문자열로 반환

push(값) : 배열 맨 뒤에 요소 추가

pop() : 배열 맨 뒤에 요소 제거

unshift(값) : 배열 제일 앞에 요소 추가

shift() : 배열 제일 앞에 요소 제거

slice(숫자,숫자) : 배열에서 지정한 범위의 값을 복사하여 리턴

splice(숫자,숫자,[값]) : 배열 범위의 값을 제거하고, 새로운 값 추가
