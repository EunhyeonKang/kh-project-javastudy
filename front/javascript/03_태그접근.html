작성된 HTML 태그 접근

엘리먼트 객체 정보 가지고오기

1. id속성 사용 : document.getElementById("아이디");

태그의 아이디 속성을 사용해서 태그정보를 가져오며,id속성은 태그의 유일한 식별자로 동일한 아이디를 여러 태그에 부여하더라도 첫번째 태그만 가져올 수 있음

2. class속성 사용 : document.getElementsByClassName("클래스")

태그의 클래스 속성을 사용해서 태그정보를 가져오며,동일한 클래스 속성값을 가진 앨리먼트를 모두 가져오며, 배열형태로 리턴
단, 해당 클래스를 가진 앨리먼트가 1개여도 무조건 배열로 리턴

3. name속석 사용 : document.getElementsByName("name속성값")

태그의 name속성을 사용해서 태그정보를 가져오며, 동일한 name속성 값을 가진 앨리먼트를 모두 가져옴

4. 태그사용 : document.getElementsByTagName("태그이름")

태그의 이름을 이용해서 태그정보를 가져옴, 동일한 태그를 모두 가져오며 앨리먼트 배열형태로 리턴

앨리먼트 객체정보사용

1. innerHTML : 앨리먼트 객체의 시작태그와 종료태그 사이의 내용

앨리먼트 내용을 읽어올 수 있으며 변경도 가능

2. outerHTML : 앨리먼트 객체의 시작태그와 종료태그를 사이의 나용을 태그를 포함해서 가져옴

앨리먼트 내용을 읽어올 수 있지만 변경은 불가능

3. value : 앨리먼트 객체의 value속성 값(주로 input태그에 사용)

앨리먼트의 value값을 읽어올 수 있으며 변경 가능
