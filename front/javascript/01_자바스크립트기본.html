자바스크립트 개요

자바스크립트 작성 방식

inline 방식 : 자바스크립트 양이 한줄 혹은 두줄정도로 소량일때 사용

internal 방식 : html 파일 내부의 자바스크립트 소스 작성(script태그 내부에 내용 작성)

external 방식 : 자바스크립트 파일을 따로 외부에 작성한 후 불러오는 방식

데이터를 출력하는 방법

1. docoment.write(내용) : 브라우저 화면상의 값을 출력

2. window.alert(내용) : 내용을 메세지창에 출력(window 생략 가능)

3. console.log(내용) : 개발자도구 화면의 콘솔에 출력(개발도중 값 확인을 목적으로 사용)

1. window.confirm("질문") : 질문에 대해 예/아니오의 결과를 얻을 때 사용(true/false)

2. window.prompt("질문") : 질문에 대해 입력할 텍스트 필드와 확인/취소 버튼이 있는 대화창 출력
