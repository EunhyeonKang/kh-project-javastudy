2. 숫자 : 정수형숫자 부동소수점 숫자 처리

nukmber 타입으로 Math라는 내장객체 사용 가능

3. 기타자료형 : 논리형(true/false), 객체(object), undefined(식별되지 않음),함수(function), 배열(Array)

자바스크립트 연산자는 자바연산자와 대부분 유사하지만 추가연산자가 존재

===, !== : 자료형과 같이 모두 일치하는지 확인하는 연산자

typeof() : 해당하는 값이 어떤 자료형을 갖는지 확인하는 연산자
