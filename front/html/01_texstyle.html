글꼴관련 스타일

front-family : 텍스트의 글꼴을 지정할때 사용

선택자{front-family : 글꼴이름}

글꼴 변경 테스트1(궁서체)

글꼴 변경 테스트2(고딕체)

글꼴 변경 테스트3(없는글꼴)

** 웹폰트 사용법

웹 폰트 제공 사이트 : 구글 웹폰트 사이트

글꼴 변경 테스트4(웹폰트)

font-size : 텍스트의 크기를 변경할 때 사용

선택자{font-size: 크기;}의 형식으로 사용
  • 크기 테스트 1(고정크기)
  • 크기 테스트 2(가변크기)

font-weight : 텍스트 두께를 표현할 때 사용

선택자{font-weight : normal | bold | bolder | lighter | 100~900} (기본값 : nomal,400)
  • 두께 테스트1(기본)
  • 두께 테스트2(가늘게)
  • 두께 테스트3(굵게)
  • 두께 테스트4(더굵게)

font-variant : 작은 대문자로 변경 떄 사용

선택자{font-variant : 속성값}

I LOVE YOU

i love you

font-style 속성 : 텍스트의 기울임을 지정할 때 사용

선택자 {font-style : normal | italic | oblique}
  • italic 기울임
  • oblique 기울임

font 속성 : 글꼴관련 스타일을 한번에 지정할때 사용

선택자{font : font-style font-variant font-weight font-size/line-height font-family;}(font-size, font-family 두가지는 필수 나머지는 안적어도 기본값적용)
  • 글꼴 관련 스타일 테스트1
  • 글꼴 관련 스타일 테스트2
