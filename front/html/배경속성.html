배경스타일

background-color : 요소의 배경색 지정

선택자 {background-color : 색상표현}

background-clip : 배경 범위 조절

선택자 {background-clip : border-box | padding-box | content-box}

background-image : 요소의 배경에 이미지를 저장

선택자 {background-image : url(이미지파일경로)}

background-repeat : 배경이미지 반복출력 설정

선택자{background-repeat : repeat | repeat-x | repeat-y | no-repeat}

background-position : 배경의 이미지의 위치를 조정하는 속성

선택자{background-position : 수평위치 수직위치}

background-size : 배경 이미지 크기를 조절하는 속성

선택자{background-size : auto| contain | cover| 입력값}

background-origin : 배경이미지 배치 시 기준

선택자{background-origin : border-box | padding-box | content-box}

background-attachement : 웹페이지 스크롤 이동 시 배경이미지 고정 여부

선택자{backround-attachement : scroll | fixed}
